# Gosia Sinilo — Portfolio

Personal portfolio website showcasing my projects, skills, and background as a Junior Full Stack Developer.

🌐 **Live site**: [gosiasinilo.com](https://gosiasinilo.com)

---

## About

Built with plain HTML, SASS/CSS, and vanilla JavaScript — no frameworks. Designed to be fast, clean, and easy to maintain.

## Features

- Responsive layout — mobile and desktop
- Project showcase with links to live deployments
- Skills section
- About / bio section with CV download
- Contact form (powered by Netlify Forms)

## Tech

- HTML5
- SASS / CSS3
- JavaScript (vanilla)
- Netlify (hosting + forms)
